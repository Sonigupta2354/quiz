Introduction

This chapter explains about Less Nesting, its syntax and gives an idea about simple and complex Nesting rules.


Description

  • Variables are shared across the imports.
  • Easy to collaborate with others.
  • The front-end development can have a modular and dry approach


Example

Following is a simple example that explains the use of imports in LESS CSS.


Step 1.

Create HTML and LESS files as shown below. Make sure that both the files are created in the Nodejs folder.
Import.html
<html>
<head>
<title>Less Importing</title>
<link rel="stylesheet" type="text/css" href="style.css">
</head>
<body>
<h1>Less Css Importing benefits</h1>
<p class="myclass">Variables are shared across the imports.</p>
<p class="myclass1">Easy to collaborate with others.</p> <p class="myclass2">The front-end development can have a modular and dry approach.</p>
</body>
<</html>


Step 2.

Now, create two .less files as shown below.
sample.less
h1{
color: #08088A;
}
.myclass{
color: #6E6E6E;
}
.myclass1{
color: #6E6E6E;
}
style.less


@import “<a href="C:\Users\saikiran chenagoni\less\sample.less"> </a >”;
.myclass2
{
Color: #01A9DB;
}


Step 3.

Compile the above codes in command prompt using the below command.
lessc style.less style.css


Step 4.

By compiling the above .less file, it automatically generates .css file as shown below.
style.css
h1 {
color: #08088A;
}
.myclass {
color: #6E6E6E;
}
.myclass1 {
color: #6E6E6E;
}
.myclass2 {
Color: #01A9DB;
}


Result

After completing all the required changes, run the html file in a browser, so that one can get the following output
LESS variables overview

Welcome to Education Portal

Variables are shared across the imports.

Easy to collectate with others.

The front-end development can have a modular and dry approach.



Key Points

  • Multiple files can be imported using @import command whenever needed.
  • The major benefit of imports is it can easily collaborate with each other.


Programming


Tips
  • Make sure to create the files in Node js folder.
  • Make sure to give the correct path in the code to import the required file
