Introduction

This chapter demonstrate about the HTML Basics which is used to define frequently used tags to develop a web pages. Following are the concepts covered.
  • Fundamental Concepts
  • Basic Tags

Description

In order to build the document or web page user need to use the three basic fundamental blocks as shown below.
  • Elements
  • Attribute
  • Content


The line below demonstrate the basic fundamental blocks of html.
<a href=”Educationportal.com”>educationportallessons</a>
From the above line.
  • <a> </a> is Elements
  • href="educationportallessons" is Attribute
  • splessons is Content of the web page.


Description

Designing an HTML page is very simple. Every HTML documents must start with document type declaration i.e. <!DOCTYPE html> The documents of HTML contains all tags between <HTML> </HTML> tag only.
The another important tag in HTML document is <body> < /body >. Visibility of the web page is enclosed in this tag only.
